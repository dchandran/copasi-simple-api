Detailed Description

Simple C API for the Copasi C++ library.

This is a C API for the COPASI C++ library. Rate equations in COPASI require the "complete name", e.g. instead of X, the rate must specify <model.compartment.X>. In this C API, those complete names are stored in a hash table. The API replaces the simple strings, i.e. "C", with the complete names by using the hash-table. This is mainly for speed; otherwise, every cSetReactionRate would be searching through the entire model for each of its variables. The hash-table idea is used for functions such as cSetValue, which can set the value of a parameter or that of a molecular species. Again, it uses the hash table to identify what a variable is.

The C API hides the C++ classes by casting some of the main classes into void pointers inside C structs.

std::map is used for performing the hashing (it is not a real hash-table, but close enough). boost::regex is used for string substitutions.
